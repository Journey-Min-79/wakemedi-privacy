1. 기획 수정 요약

기존 v1.0 기획서는 스트레스 지수 + 회복 점수 + 로컬 저장 + 사용자 직접 AI 선택 방향이 잘 잡혀 있었다. v1.1에서는 실제 출시를 고려하여 아래 내용을 보완한다.

구분기존 방향보완 방향
앱 포지션로컬 기반 스트레스 지수 & AI 컨디션 코치스트레스 지수를 더 전면에 배치하고, 회복/수면/감정/AI를 보조 기능으로 정리
수익화Pro 중심무료 앱 진입을 위해 Google AdMob 광고 추가, Pro는 광고 제거와 고급 분석 중심
광고 정책건강 데이터 기반 광고 없음AdMob 광고는 허용하되 건강 데이터·감정 기록·AI 내용은 광고 타겟팅에 사용하지 않는 원칙 명확화
화면 구성오늘/기록/트렌드/AI/설정광고 배치 가능 화면과 광고 제외 화면을 분리
개인정보로컬 저장 중심광고 SDK, AI Provider, HealthKit/Health Connect 데이터 흐름을 분리해서 설명
심사 대응의료 면책 문구 중심Apple/Google 건강 데이터 정책, 광고 정책, 개인정보 처리방침 대응 항목 추가

2. 앱 개요

항목내용
앱 유형iOS / Android 건강 컨디션 분석 앱
개발 방식Flutter 기반 크로스플랫폼 앱
건강 데이터 연동iOS: Apple HealthKit / Android: Health Connect
서버 사용자체 서버 없음
저장 방식기기 내부 로컬 DB 저장
AI 사용 방식사용자가 직접 AI 제공자와 API Key를 등록하여 사용
광고 사용Google AdMob 적용
광고 정책건강 데이터 기반 타겟팅 금지, 민감 화면 광고 최소화
핵심 지표스트레스 지수
보조 지표회복 점수, HRV, 안정 시 심박수, 수면, 운동, 걸음, 감정 기록
주요 사용자스트레스 관리, 수면 관리, 운동 컨디션 관리에 관심 있는 일반 사용자

이 앱은 건강 데이터를 바탕으로 사용자의 오늘 스트레스 지수를 계산하고, 그 원인과 회복 방향을 보여주는 앱이다. 단순히 점수만 제공하지 않고, 수면·운동·감정 기록·생활습관을 함께 분석하여 사용자가 오늘 어떤 행동을 해야 하는지 판단할 수 있게 돕는다.

앱 자체 서버는 두지 않는다. 건강 데이터, 감정 기록, 분석 결과는 기본적으로 기기 내부에 저장한다. 단, 사용자가 AI 기능을 켜거나 광고가 노출될 경우 외부 서비스와의 통신이 발생할 수 있으므로, 아래처럼 데이터 흐름을 명확히 분리한다.

구분외부 통신 여부설명
건강 데이터 조회없음HealthKit / Health Connect에서 기기 내 권한 기반 조회
로컬 점수 계산없음앱 내부 규칙 엔진에서 계산
감정 기록 저장없음로컬 DB 저장
AI 코치있음사용자가 선택한 AI 제공자에게 요약 데이터 직접 전송 가능
Google AdMob 광고있음광고 로딩을 위해 Google 광고 SDK 통신 발생
자체 서버없음회원가입, 서버 저장, 서버 분석 없음

3. 앱 콘셉트

오늘 내 몸의 스트레스 신호를 확인하고, 어떻게 쉬고 움직일지 알려주는 로컬 기반 컨디션 앱

이 앱의 중심은 스트레스 지수다. 사용자가 앱을 열었을 때 가장 먼저 확인하는 것은 “오늘 스트레스가 높은가?”이며, 그 다음에 “왜 높아졌는가?”, “오늘 무엇을 하면 좋은가?”로 이어진다.

구분기획 방향
메인 메시지오늘 내 몸이 얼마나 부담을 받고 있는지 확인
핵심 지표스트레스 지수 0~100
보조 지표회복 점수, 수면 상태, 운동 부하, 감정 기록
핵심 행동오늘 무리할지, 쉬어야 할지, 가볍게 움직일지 판단
차별점서버 없음, 사용자 직접 AI 연결, 감정 기록과 건강 데이터 연결
수익화무료 사용 가능 + 광고 노출 + Pro 광고 제거

4. 사용자 문제와 앱의 해결 방향

4-1. 사용자 문제

사용자 문제설명
스트레스를 느끼지만 객관적으로 확인하기 어렵다피곤하고 예민하지만 몸 상태가 실제로 나쁜지 알기 어렵다
수면, 운동, 심박 데이터가 흩어져 있다건강 앱에 데이터는 있지만 해석이 부족하다
오늘 운동해도 되는지 판단이 어렵다컨디션이 안 좋은 날 강한 운동을 하거나, 반대로 움직여도 되는 날 쉬어버린다
감정과 몸 상태의 관계를 알기 어렵다업무 스트레스, 수면 부족, 카페인 등이 몸에 어떤 영향을 주는지 모른다
개인정보가 민감하다건강 데이터와 감정 기록을 서버에 저장하는 앱은 부담스럽다
AI 조언은 받고 싶지만 데이터 통제가 필요하다어떤 AI로, 어떤 데이터를 보낼지 사용자가 선택하고 싶다

4-2. 해결 방향

해결 방향설명
스트레스 지수 제공HRV, 심박, 수면, 운동, 감정 기반으로 오늘 점수 제공
원인 분석 제공스트레스 지수가 높아진 이유를 1~3개로 설명
추천 행동 제공휴식, 산책, 카페인 조절, 수면 루틴 등 행동 제안
감정 기록 제공사용자가 직접 기분과 원인을 기록
로컬 저장 제공건강 데이터와 기록을 기기 내부에 저장
AI 선택권 제공사용자가 직접 AI Provider와 API Key를 등록
무료 사용 제공광고 기반 무료 이용 가능

5. 핵심 가치 제안

5-1. 사용자에게 보여줄 한 줄 가치

오늘 스트레스 지수를 확인하고, 내 몸에 맞는 회복 행동을 추천받으세요.

5-2. 앱스토어용 소개 문구 초안

BodySignal은 건강 데이터와 감정 기록을 바탕으로
오늘의 스트레스 지수와 회복 상태를 알려주는 컨디션 앱입니다.

HRV, 안정 시 심박수, 수면, 운동, 걸음 데이터를 로컬에서 분석하고,
사용자가 직접 기록한 감정과 생활 습관을 함께 확인할 수 있습니다.

회원가입 없이 사용할 수 있으며,
AI 기능은 사용자가 직접 선택한 AI와 API Key로만 동작합니다.

5-3. 마케팅 핵심 문구

문구사용 위치
오늘 스트레스 지수를 한눈에스토어 스크린샷 1번
왜 피곤한지 원인까지 확인스토어 스크린샷 2번
수면·운동·감정 기록을 함께 분석스토어 스크린샷 3번
서버 없이 로컬 중심으로 저장개인정보 강조 화면
내가 선택한 AI로 컨디션 조언 받기AI 기능 화면
무료로 시작하고, 광고 제거는 선택수익화 안내 화면

8. 스트레스 지수 기획

8-1. 기본 정의

스트레스 지수는 사용자의 건강 데이터와 감정 기록을 바탕으로 산출되는 컨디션 참고 지표다. 의료 진단이나 질병 예측이 아니라, 오늘 몸이 평소보다 얼마나 부담을 받고 있는지 보여주는 앱 내부 점수로 정의한다.

점수 구간상태사용자 표현
0~25안정몸이 편안한 상태입니다
26~50보통평소와 비슷한 상태입니다
51~75주의몸이 부담을 느끼는 신호가 있습니다
76~100과부하오늘은 회복을 우선하는 것이 좋습니다

8-2. 표현 가이드

사용 가능 표현피해야 할 표현
회복 신호가 낮습니다자율신경 이상입니다
몸이 평소보다 부담을 느낄 수 있습니다질병 위험이 있습니다
오늘은 무리하지 않는 것이 좋습니다치료가 필요합니다
강한 운동보다 가벼운 활동을 추천합니다우울증 가능성이 있습니다
수면과 휴식을 우선해보세요심장 이상이 의심됩니다

8-3. 계산 원칙

스트레스 지수는 절대값이 아니라 개인 기준값 대비 변화량을 기준으로 계산한다.

예를 들어 HRV 30ms는 모든 사용자에게 나쁜 값이 아니다. 평소 HRV가 60ms인 사용자가 30ms로 떨어졌다면 스트레스 신호가 강하지만, 평소 25ms인 사용자에게 30ms는 안정적인 값일 수 있다.

8-4. 기본 계산 요소

요소기본 비중설명
HRV 변화35%최근 7일/30일 평균 대비 HRV 하락 여부
안정 시 심박수20%평소보다 안정 시 심박수가 높아졌는지
수면 상태20%수면 시간, 수면 규칙성, 중간 깨어남 여부
운동 / 활동 부하15%과도한 운동 또는 활동 부족 여부
감정 기록10%부정 감정, 스트레스 원인 태그, 강도

0~39회복 매우 부족강한 운동, 야근, 과로 주의

24. 최종 추천 방향

BodySignal — 스트레스 지수를 중심으로 한 로컬 기반 AI 컨디션 코치

24-1. 최종 앱 정의

BodySignal은 HRV, 심박, 수면, 운동, 감정 기록을 바탕으로
오늘의 스트레스 지수와 회복 상태를 알려주는 Flutter 기반 iOS/Android 앱이다.

앱 자체 서버 없이 로컬 저장을 기본으로 하며,
사용자가 직접 선택한 AI와 API Key를 통해 컨디션 조언을 받을 수 있다.

무료 사용자는 일부 화면에서 Google AdMob 광고를 볼 수 있고,
Pro 사용자는 광고 없이 고급 분석 기능을 사용할 수 있다.

24-2. 핵심 기능 5개

핵심 기능설명
스트레스 지수HRV, 심박, 수면, 운동, 감정 기반 0~100 점수
회복 점수오늘 무리해도 되는지 알려주는 회복 상태
원인 분석왜 스트레스가 높아졌는지 설명
감정 기록사용자가 직접 감정과 원인 기록
AI 코치사용자가 직접 등록한 AI로 조언 받기

24-3. 수익화 핵심

구분방향
무료오늘 지수, 7일 트렌드, 감정 기록, 광고 표시
Pro광고 제거, 월간/연간 분석, AI 리포트, 데이터 내보내기
광고건강 데이터 기반 타겟팅 금지, 민감 화면 미노출
AI사용자 API Key 방식으로 앱의 AI 비용 부담 없음

24-4. 기획 요약

스트레스 지수를 메인으로 보여주되, 스트레스만 보여주지 않는다.

HRV, 심박, 수면, 운동, 감정 기록을 함께 분석한다.
사용자는 오늘 왜 피곤한지, 운동해도 되는지, 쉬어야 하는지 알 수 있다.

서버는 없다.
건강 데이터는 로컬에 저장한다.
AI는 사용자가 직접 선택하고 직접 API Key를 넣는다.

무료 사용자는 광고를 통해 부담 없이 시작할 수 있고,
Pro 사용자는 광고 없이 더 깊은 분석을 사용할 수 있다.

앱의 핵심 가치는 "스트레스 점수"가 아니라
"오늘 내 몸을 어떻게 다뤄야 하는지 알려주는 것"이다.
